goblog
======

Learn to write a blog framework from scratch in golang

**Installation**

- $ cd $GOPATH
- $ go get github.com/ouyanggh/goblog 
- $ cd src/github.com/ouyanggh/goblog
- $ go run main.go

**Usage**

- enter: http://localhost:8080/
- admin portal: http://localhost:8080/admin  admin/hello
